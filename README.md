# Jimmy

Jimmy is a WSJT-X companion app designed for improved and simplified accessibility for blind amateur radio operators.

## Requirements

- Windows 10/11
- WSJT-X 2.6 or later
- .NET Framework 4.7.2

## Installation

Download the latest `Jimmy.msi` from the [Releases](https://github.com/jimr9/Jimmy/releases) page and run it.

## Building from Source

Open `Jimmy.sln` in Visual Studio 2022 and build in Release mode.

To build the installer: `wix build -o Release\Jimmy.msi Jimmy.wxs` (from `Setup_WiX\`)
